
Excel to SQL Data Migration

Overview
This project demonstrates migrating structured sales data
from Excel/CSV format into a MySQL database using Python.

Tools Used
- Python
- Pandas
- MySQL
- mysql-connector

Dataset
Sample sales dataset in CSV format (simulating Excel input).

Migration Steps
1. Loaded Excel/CSV data using Pandas
2. Removed duplicate and incomplete records
3. Designed normalized SQL table
4. Inserted records using Python connector
5. Verified data integrity

Output
Cleaned data successfully migrated to MySQL database.

Notes
This project was tested on a local MySQL environment.
